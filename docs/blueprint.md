# **App Name**: Mausam India

## Core Features:

- Location Search: Allows users to search for current weather conditions for various Indian cities.
- 3-Day Forecast: View a 3-day forecast for the selected location.
- Detailed Weather Information: Display weather details like humidity and wind speed.
- AI Weather Assistant: An AI powered assistant to explain in natural language weather insights and their impact. It acts as a 'tool' to give simplified details to users. 

## Style Guidelines:

- Primary color: Vibrant sky blue (#40A4DF) to evoke a sense of clear skies and daytime weather.
- Background color: Desaturated sky blue (#E0F0FB) for a calming backdrop.
- Accent color: Sun orange (#FFA500) to represent sunshine and warmth, used for highlights and important information.
- Body and headline font: 'PT Sans', a sans-serif for a clean and modern look, suitable for headlines or body text.
- Use clear, representative icons for different weather conditions (sun, rain, clouds).
- Clean, intuitive layout with a focus on easy access to key weather information.
- Subtle animations for weather changes and data updates.
